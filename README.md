# Naman's Portfolio

A cutting-edge, single-page portfolio showcasing work at the intersection of Hardware & Intelligence.

## 🚀 Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling with Linear-esque design
- **Framer Motion** - Smooth animations and transitions
- **Lucide React** - Beautiful icon set

## 🎨 Design Features

- **Ultra-dark theme** (#050505 background)
- **Glassmorphism effects** with hover animations
- **Rotating role tags** in hero section
- **Bento grid layout** for projects
- **Smooth scroll** and staggered animations
- **Fully responsive** mobile-first design

## 📦 Installation

```bash
npm install
```

## 🏃 Running Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🏗️ Build for Production

```bash
npm run build
npm start
```

## 📁 Project Structure

```
├── app/
│   ├── layout.tsx       # Root layout with metadata
│   ├── page.tsx         # Main page component
│   └── globals.css      # Global styles
├── components/
│   ├── Hero.tsx         # Hero section with rotating roles
│   ├── Projects.tsx     # Project bento grid
│   ├── Skills.tsx       # Technical arsenal
│   ├── Experience.tsx   # Timeline component
│   ├── Contact.tsx      # Contact section
│   └── Footer.tsx       # Footer component
├── constants/
│   └── index.ts         # All hardcoded data
└── public/              # Static assets
```

## 🎯 Sections

1. **Hero** - Name, headline, rotating roles, and bio
2. **Projects** - Bento grid showcasing CivicLensAI, ChatbotJarvis, MindCareAI, and Third Eye
3. **Technical Arsenal** - Skills grouped by Software, Intelligence & Logic, and Systems
4. **Experience** - Timeline of internships at Lenovo and LearnTube.ai
5. **Contact** - Social links and contact information
