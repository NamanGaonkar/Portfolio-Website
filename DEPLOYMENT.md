# 🚀 Deploy to Vercel

## Quick Deploy (Right Now!)

Your portfolio is **ready to deploy** without any images! The project cards have beautiful gradient fallbacks.

### Option 1: Deploy via Vercel CLI

```bash
npm install -g vercel
vercel
```

### Option 2: Deploy via Vercel Dashboard

1. Go to [vercel.com](https://vercel.com)
2. Click "Import Project"
3. Select your GitHub repository (or upload this folder)
4. Vercel will auto-detect Next.js and deploy!

## 📸 Adding Images Later (Optional)

Just drop your project images into the `public/projects/` folder:

```
public/
  projects/
    civiclens.jpg    ← CivicLensAI image
    jarvis.jpg       ← ChatbotJarvis image  
    mindcare.jpg     ← MindCareAI image
    vitemate.jpg     ← Vitemate image
```

**Image specs:**
- Aspect ratio: 16:9 (1920x1080 or 1280x720)
- Format: JPG, PNG, or WebP
- Size: Keep under 500KB each

Then just push to GitHub and Vercel will auto-redeploy!

## ✅ Site Works Without Images

The gradient backgrounds look great as-is, so you can deploy immediately and add images anytime later. No blockers! 🎉
